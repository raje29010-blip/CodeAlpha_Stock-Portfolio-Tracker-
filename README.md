# 📈 Stock Portfolio Tracker

A simple Python-based Command Line Interface (CLI) application to track stock investments, calculate total value, and save results to a file.

---

## 🚀 Features
- Track multiple stock investments
- Calculates total investment automatically
- User-friendly CLI interface
- Validates stock input
- Saves output to a file (`portfolio.txt`)

---

## 🛠️ Technologies Used
- Python 3
- File Handling
- Dictionary
- Loops & Conditions

---

## ▶️ How to Run
1. Install Python
2. Run the file:
```bash
python stock_portfolio.py
